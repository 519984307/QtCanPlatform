Einleitung
----------
PLIN steht für PEAK LIN Applications und ist ein System für die Entwicklung
und zum Betrieb von LIN-Bussen. Es ist ein hilfreiches und umfassendes Produkt,
das sowohl an Entwickler als auch an Endbenutzer gerichtet ist.
Das PLIN-System erlaubt eine Echtzeitanbindung von Windows-Anwendungen an
LIN-Busse, die über PCAN-USB Pro Adapter physikalisch an den PC angeschlossen
sind.

Die PLIN-API ist eine Programmier-Schnittstelle zum PLIN-System. Über eine
Interface-DLL wird die Verbindung eigener Applikationen an den Gerätetreiber
und den PCAN-USB Pro Adapter ermöglicht, um eine Kommunikation mit LIN-Bussen
zu betreiben.


Systemvoraussetzungen
---------------------
- Betriebssysteme: Windows 7/Vista/XP
- Ein freier USB Anschluss (empfohlen USB 2.0) am Computer
- PCAN-USB Pro Adapter

Installation
------------
Damit die PLIN-API mit dem angeschlossenen PCAN-USB Pro Adapter kommunizieren kann,
muss das PCAN-USB Pro Setup installiert werden. Dieses befindet sich auf der
Produkt-CD im Verzeichnis <Produkt-CD>\Tools\PCAN-USB Pro

Für weitere Informationen über die Inbetriebnahme des PCAN-USB Pro Adapters beachten
Sie bitte die Dokumentation der Hardware. Diese finden Sie im Verzeichnis
<Produkt-CD>\Pdf
